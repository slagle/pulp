Package Group Categories

A package group category is a collection of package group ids
Once you get a repository created you can start managing package group categories. Package group categories exist as members of a repository.
Note: PackageGroupCategory operations are served from the repository controller

List

List the package group categories under a repository
See UGREST-Repositories#RepositoryPackageGroupCategories

Create

Create a new packagegroup
See UGREST-Repositories#CreatePackageGroupCategory

Delete

Delete a package group category
See UGREST-Repositories#DeletePackageGroupCategory

Add a Package Group

Add a package group to a package group category
See UGREST-Repositories#AddPackageGroupToCategory

Delete a Package Group

Delete a package group from a package group category
See UGREST-Repositories#DeletePackageGroupFromCategory

Install a Package Group Category

Install a package group category on a consumer
See UGREST-Consumers#InstallPackageGroupCategory
