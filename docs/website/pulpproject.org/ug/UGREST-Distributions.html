Distribution RESTful API

No description.

Look up distribution by id.

Get a distribution object

method: GET

path: /distributions/<id>/

permission: READ

success response: 200 OK

failure response: 404 Not Found if the id does not match a distribution

return: a Distribution object

example:

List all available distributions.

Get a list of all distributions managed by Pulp.

method: GET

path: /distributions/

permission: READ

success response: 200 OK

failure response: None

return: list of distribution objects, possibly empty

example:
