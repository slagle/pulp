Package Groups

Once you get a repository created you can start managing package groups.
Package groups exist as members of a repository.
Note: PackageGroup operations are served from the repository controller

List

List the package groups under a repository
See UGREST-Repositories#RepositoryPackageGroups

Create

Create a new package group
See UGREST-Repositories#CreateAPackageGroup

Delete

Delete a packagegroup
See UGREST-Repositories#DeleteAPackageGroup

Add Package

Add a package to a package group
See UGREST-Repositories#AddPackagesToPackageGroup

Delete Package

Delete a package from a package group
See UGREST-Repositories#DeleteAPackageGroup

Import

Import a comps.xml and create groups and categories
See UGREST-Repositories#ImportComps

Export

Export a comps.xml for a repository
See UGREST-Repositories#RepositoryCompsXML

Install

Install a package group on a consumer
See UGREST-Consumers#InstallPackageGroup
Install a package group on a consumer group See UGREST-ConsumerGroups#InstallPackageGroups

Uninstall

Uninstall a package group on a consumer
See UGREST-Consumers#UninstallPackageGroup
Uninstall a package group on a consumer
See UGREST-ConsumerGroups#UninstallPackageGroups
