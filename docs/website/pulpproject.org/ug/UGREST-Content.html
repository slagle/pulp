Content

Pulp exposes a content API to perform file specific operations.

Base URL Path: /content/

Find Content

lookup files on pulp server by id

method: GET

path: /content/file/<id>

permission: READ

success response: 200, OK

failure response: 404, Not Found

return: <dict>

  • id <str> the content id
  • checksum <str> the content checksum
  • description <str> the content description
  • download_url <str> the file download URL
  • filename <str> the name of the file containing the content as stored on the server
  • repo_defined: <bool> indicates the content belongs to a repository
  • size <int> the content file size in bytes

sample response: 

{ 'id': 'c5b1dd1e-2770-416f-8966-1c7c4a8832e7',
  'checksum': {'sha256': 'e3b0c44298fc1c149afbf4c8996fb92427ae41e4649b934ca495991b7852b855'},
  'description': None,
  'download_url': None,
  'filename': 'fake.iso',
  'repo_defined': False,
  'size': None}

Delete Content

Delete a single content file.

method: DELETE

path: /content/file/<id>/

permission: DELETE

success response: 200, OK

failure response: 404, Not Found

return: <bool> True

sample response: 

True
